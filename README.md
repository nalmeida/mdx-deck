# Custom theme and sample for mdx-deck presentations

## Install mdx-deck

```
$ npm i -g mdx-deck
```


## How to use this theme

1. Clone this repo
2. Create yout presentation
3. Run the following command to render your presentation:

``` 
$ mdx-deck start presentation.mdx
```
